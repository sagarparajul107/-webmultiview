Frequently Asked Questions

How do I add a new website to view?
Simply enter the website URL in the input field on the main dashboard and click the "Add" button. You can then select how many views you want to generate.
Is there a limit to how many websites I can track?
No, there is no limit. MultiView is completely free and you can track as many websites as you want.
How do I delete a website from my list?
Go to the dashboard, find the website you want to remove, and click the trash icon next to it. Confirm the deletion when prompted.
Does MultiView work on mobile devices?
Yes! MultiView is fully responsive and works on all modern mobile devices and browsers.
Do I need to pay for premium features?
No, MultiView is completely free. There are no premium features or subscription plans. All features are available to all users at no cost.
